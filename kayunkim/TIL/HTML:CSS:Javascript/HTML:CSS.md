#### Display:flex;

```
display:flex;
justify-content:center;  #main축 중심으로(기준: 가로, flex:direction: column;: 세로)
align-items:center;   #cross축(main의 반대) 중심으로 
```

요소 속성들이 block -> item으로 바뀜
