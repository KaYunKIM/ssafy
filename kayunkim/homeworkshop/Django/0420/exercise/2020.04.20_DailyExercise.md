## 1.

```SQL
CREATE TABLE FLIGHTS(
	ID INTEGER PRIMARY KEY AUTOINCREMENT,
  FLIGHT_NUM TEXT NOT NULL,
  DEPARTURE TEXT NOT NULL,
  WAYPOINT TEXT NOT NULL,
  ARRIVAL TEXT NOT NULL,
  PRICE INTEGER NOT NULL
);
```
